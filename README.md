# Projeto horário 

## Descrição
- Projeto básico que muda o horário exibido com base no dispositivo de usuário.
### A aplicação também muda a imagem de background baseado nos seguintes períodos do dia
- Amanhecer
- Manhã
- Meio-dia
- Tarde
- Anoitecer
- Noite / Madrugada

## Técnologias
- JavaScript
- HTML
- CSS
